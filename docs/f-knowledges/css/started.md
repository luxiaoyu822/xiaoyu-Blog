# CSS 基础

`CSS`（Cascading Style Sheets，层叠样式表）是为 web 内容添加样式的代码。
本节将介绍 `CSS` 的基础知识，并解答像这样的问题：怎样将文本设置为红色？怎样
将内容显示在屏幕的特定位置？怎样用背景图片或颜色来装饰网页？
